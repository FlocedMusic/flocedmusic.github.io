# Floced website

Site statique pour `https://flocedmusic.github.io/`, pret a publier avec GitHub Pages.

## Fichiers

- `index.html` : page principale
- `styles.css` : design responsive noir, blanc et rouge
- `script.js` : menu mobile et en-tete au scroll
- `assets/` : visuels du site

La rubrique `Atelier` est prevue pour recevoir les photos du materiel utilise :
machines analogiques, cables jack confectionnes, boites passives, transfos audio
prototypes et reparations.

## Publication GitHub Pages

1. Envoyer ces fichiers dans le depot GitHub `flocedmusic.github.io`.
2. Dans GitHub, ouvrir `Settings > Pages`.
3. Choisir la branche principale et le dossier racine.
4. Le site sera disponible sur `https://flocedmusic.github.io/`.

## Domaine Floced.fr chez OVH

Quand le domaine sera pret, ajouter dans GitHub Pages le domaine `floced.fr`, puis creer les entrees DNS recommandees par GitHub dans l'espace OVH. A ce moment-la seulement, ajouter un fichier `CNAME` contenant :

```text
floced.fr
```
